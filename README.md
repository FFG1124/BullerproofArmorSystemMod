本模组为Minecraft引入了基于等级的完整甲弹对抗系统，让战斗更加真实和富有策略性。

核心机制
护甲系统
护甲分为1-6级，每级提供10%伤害减免，最高60%。采用部位伤害判定系统，根据攻击者高度差区分头部、胸部、腿部等命中部位。

弹药与武器系统
弹药和武器同样分为1-6级，每提升一级增加20%伤害加成（最高+100%）。弹药等级越高，对高等级护甲的穿透能力越强。

耐久系统
护甲被击中后根据承受伤害和自身等级扣除耐久，高级护甲更耐打。

内置默认配置
原版护甲等级：

皮革套：1级（10%减伤）

锁链套/金套：2级（20%减伤）

铁套：3级（30%减伤）

钻石套：4级（40%减伤）

下界合金套：5级（50%减伤）

原版武器等级：

木质：1级（+0%）

石质：2级（+20%）

铁质：3级（+40%）

钻石：4级（+60%）

下界合金：5级（+80%）

支持的模组集成
TACZ（永恒枪械工坊）：21种弹药类型预设及弹射物伤害处理

Delta模组：30+种弹药等级配置

Tarkov模组：50+种护甲等级配置（胸甲、头盔、安全箱）

使用方法
图形界面
按下 F8 键打开配置界面，可视化设置护甲、弹药、武器等级。

命令系统（/bas）
命令	说明

/bas armor hand	查看手持护甲信息

/bas armor set <1-6>	设置手持护甲等级

/bas ammo hand	查看手持弹药信息

/bas ammo set <1-6>	设置手持弹药等级

/bas weapon hand	查看手持武器信息

/bas weapon set <1-6>	设置手持武器等级

/bas list	列出所有已配置物品

/bas reload	重载配置文件

/bas debug	调试护甲状态

配置文件
位于 .minecraft/config/bullerproof_armor_system/：

armor_tiers.json - 护甲等级

ammo_tiers.json - 弹药等级

weapon_tiers.json - 武器等级

适用场景
PVP服务器、枪械模组整合包、RPG服务器、军事模拟玩法

注意事项
本模组需要 Forge 1.20.1 版本，与 TACZ、Delta、Tarkov 等模组有良好的兼容性。
